ARTVEE 이미지 다운로드 프로그램 사용 가이드

■ 1. 프로그램 기본 폴더 구성

프로그램을 실행하기 전에 아래와 같은 기본 폴더 구조가 필요합니다.


프로그램폴더/
    ├─ main.exe
    └─ result/
        ├─ excel/
        └─ image/


폴더	설명
main.exe	    프로그램 실행 파일
result/excel	수집된 데이터 엑셀 파일 저장 위치
result/image	다운로드된 이미지 저장 위치

※ 프로그램 실행 시 필요한 엑셀 파일과 이미지 파일은 위 폴더 안에 자동으로 저장됩니다.


■ 2. 프로그램 실행

프로그램 실행 시 아래 메뉴가 표시됩니다.

1. artvee 다운
2. 엑셀 번역
3. artvee artist 다운
4. collection by category 선택
5. 이미지 수 확인

각 메뉴 기능은 아래와 같습니다.



3. 메뉴 기능 설명
   ① artvee 다운

전체 작가 데이터를 수집하고 이미지를 다운로드하는 기능입니다.

실행 과정

1️⃣ 프로그램 실행

1. artvee 다운

선택

2️⃣ 다음 질문이 표시됩니다.

전체 엑셀 추출 하시겠습니까?
1. 예
2. 아니오

설명

선택	기능
1	전체 작가 데이터를 다시 수집
2	기존 엑셀 데이터를 사용

3️⃣ 다음 질문이 표시됩니다.

1. 이미지 다운로드
2. 다운안된 이미지 재 다운로드

설명

선택	기능
1	모든 이미지를 다운로드
2	실패한 이미지만 재다운로드
결과

프로그램 실행 후 아래 위치에 저장됩니다.

result/excel/artvee_1.xlsx
result/image/

이미지는 작가별 폴더로 저장됩니다.

예시

result/image/
├─ 1_Monet/
├─ 2_Van Gogh/
└─ ...
② 엑셀 번역

이미 수집된 엑셀 파일의 작품명을 한국어로 번역하는 기능입니다.

번역 대상

작품명

작품명 전체

번역 결과는 엑셀에 아래 컬럼으로 추가됩니다.

번역-1(괄호포함)
번역-2(괄호 미포함)
③ artvee artist 다운

특정 작가만 다운로드할 때 사용하는 기능입니다.

사용하려면 아래 파일이 필요합니다.

result/artvee_artist_list.xlsx

엑셀 파일 형식

artist_list
https://artvee.com/artist/claude-monet/

https://artvee.com/artist/vincent-van-gogh/

프로그램은 해당 작가의 작품만 수집합니다.

④ collection by category 선택

특정 카테고리별 이미지 다운로드 기능입니다.

프로그램 실행 시 아래 카테고리가 표시됩니다.

1. Abstract
2. Figurative
3. Landscape
4. Posters
5. Illustration
6. Religion
7. Drawings
8. Mythology
9. Botanical
10. Asian Art
11. Animals

여러 개 선택 가능합니다.

예시

1,3,5

결과 저장 위치

result/excel/collection/
result/image/collection/

폴더 구조

result/image/collection/
├─ abstract/
│   ├─ category/
│   └─ artist/
├─ landscape/
└─ ...
⑤ 이미지 수 확인

다운로드된 이미지 개수를 확인하는 기능입니다.

카테고리를 선택하면 아래 두 가지 수량이 표시됩니다.

1️⃣ category 폴더 이미지 수
2️⃣ artist 폴더 이미지 수

예시

landscape 카테고리 'category' 폴더 내 이미지 수: 1520개
landscape 카테고리 'artist' 폴더 내 이미지 수: 1520개
4. 이미지 재 다운로드 방법

이미지 다운로드 중 일부 실패할 수 있습니다.

실패한 이미지는 엑셀 파일에 X 표시로 기록됩니다.

예시

이미지 저장여부 = X

실패한 이미지만 다시 다운로드하려면 아래 순서로 실행합니다.

선택

2. 다운안된 이미지 재 다운로드

선택

프로그램은 X 표시된 이미지만 다시 다운로드합니다.

5. 주의사항

1️⃣ 엑셀 파일을 열어둔 상태에서는 프로그램이 정상 작동하지 않을 수 있습니다.

엑셀파일을 닫아주세요

메시지가 나오면 엑셀을 닫고 다시 실행해 주세요.

2️⃣ 프로그램 실행 중 인터넷 연결이 필요합니다.

3️⃣ 이미지 다운로드 중 일부 이미지는 사이트에 존재하지 않아 다운로드되지 않을 수 있습니다.

6. 권장 사용 방법

일반적인 사용 순서

1️⃣ artvee 다운 실행
2️⃣ 이미지 다운로드
3️⃣ 실패 이미지 재 다운로드
4️⃣ 필요 시 엑셀 번역
