1. 분석 개요 및 목적

    유튜브 댓글은 사람들의 '진짜 속마음'과 트렌드가 가장 생생하게 담겨 있는 곳입니다.
    이 프로젝트는 수많은 댓글 속에 숨겨진 사람들의 반응을 데이터로 읽어내고,
    이를 바탕으로 모두가 공감할 수 있는 '새로운 콘텐츠'를 기획하는 것을 목표로 합니다.

    1-1. 핫한 이슈와 사람들의 반응 수집
    지금 가장 화제가 되거나 대중의 흥미를 끄는 유튜브 영상을 찾아, 그곳에 남겨진 수많은 댓글 데이터를 빠짐없이 수집합니다.

    1-2. 키워드와 감정 분석으로 진짜 여론 읽기
    단순히 눈으로 댓글을 훑어보는 것을 넘어, 데이터 분석 기술을 활용해 사람들의 전체적인 분위기를 파악합니다. 어떤 단어가 가장 핵심적으로 쓰였는지, 단어들이 서로 어떻게 꼬리를 물고 연결되어 있는지 파악하고, AI를 활용해 대중의 반응이 긍정적인지 부정적인지(감성 분석)를 객관적인 수치로 확인합니다.

    1-3. 느낌이 아닌 '데이터' 기반의 콘텐츠 제작
    "대충 반응이 이런 것 같아"라는 개인적인 추측을 배제합니다. 실제 분석으로 얻어낸 여론의 흐름과 관심사를 바탕으로, 시청자들이 진짜로 원하고 공감할 수 있는 설득력 있는 2차 콘텐츠를 만들어냅니다.


2. 데이터 수집 및 분석 방법론
    본 프로젝트는 수많은 사람들의 의견을 객관적으로 분석하기 위해, 학계와 산업계에서 널리 쓰이는 전문적인 빅데이터 기술을 적용합니다.

    2-1. 핵심 기술 : 텍스트 마이닝(Text Mining) 도입 배경

        수만, 수십만 개에 달하는 유튜브 댓글을 사람이 일일이 읽고 전체 분위기를 정확히 파악하는 것은 물리적으로 불가능하며, 개인의 주관이나 편견이 섞일 수밖에 없습니다.
        이를 해결하기 위해 본 프로젝트는 '데이터 마이닝(Data Mining)'과 '텍스트 마이닝(Text Mining)' 기법을 활용합니다.

        데이터 마이닝(Data Mining): 광산에서 흙을 걸러내고 순금만 캐내듯, 방대한 규모의 데이터 속에서 숨겨진 유의미한 패턴, 규칙, 트렌드를 수학적 알고리즘으로 찾아내는 기술입니다.
        텍스트 마이닝(Text Mining): 데이터 마이닝의 핵심 분야 중 하나로, 유튜브 댓글과 같이 형태가 일정하지 않은 글(자연어)을 컴퓨터가 이해할 수 있도록 쪼개고 분석하여 가치 있는 정보를 추출하는 고도화된 기법입니다.

        💡 왜 이 기술을 신뢰할 수 있는가?
        텍스트 마이닝은 단순히 요즘 유행하는 기술이 아닙니다. 실제 국내외 대학원의 석·박사 학위 논문 연구, 정부 기관의 사회 여론 동향 파악, 대기업의 소비자 반응(VOC) 분석 등 최고 수준의 빅데이터 연구에서 표준으로 사용되는 가장 검증된 방법론입니다.
        이 기술을 통해 "대충 이런 반응인 것 같다"는 느낌이 아니라, 객관적인 수치와 과학적 패턴으로 증명된 신뢰도 높은 여론을 읽어냅니다.


    2-2. 1단계: 구글 공식 API를 통한 데이터 수집

        정확한 분석은 깨끗하고 온전한 원본 데이터에서 출발합니다.

        YouTube Data API 활용: 화면에 보이는 댓글을 단순 복사하는 것이 아니라, 구글(Google)에서 공식적으로 제공하는 '유튜브 데이터 API(YouTube Data API v3)'를 이용해 시스템적으로 접근합니다.
        빠르고 누락 없는 수집: 영상의 조회수, 좋아요 수 같은 기본 정보는 물론, 수만 개의 원댓글과 그 아래 달린 대댓글(Reply)까지 누락 없이 빠르고 정확하게 수집합니다.
        안전한 데이터 보관: 수집된 엄청난 양의 데이터는 대용량 처리에 특화된 전문 데이터베이스(PostgreSQL)에 안전하게 저장되어 분석을 위한 '데이터 댐' 역할을 하게 됩니다.



### 3. 데이터 전처리 및 형태소 분석 (Data Preprocessing & Morphological Analysis)

수집된 데이터는 '가공되지 않은 원석'과 같습니다. 이 단계에서는 컴퓨터가 정확하게 의미를 파악할 수 있도록 불순물을 제거하고, 문장을 의미 있는 단위로 쪼개는 고도의 정제 과정을 거칩니다.

#### 3-1. 데이터 전처리 (Data Preprocessing): 분석의 기초 다지기

본격적인 분석에 앞서 데이터의 노이즈를 제거하고 일관성을 맞추는 단계입니다. 대학원이나 기업 분석 프로젝트에서 가장 많은 시간을 할애하는 중요한 공정입니다.

* **결측치 보호 처리 (Handling Missing Values):** 데이터 수집 과정에서 누락된 빈 댓글이나 오류 값이 분석 시스템을 멈추게 하지 않도록 안전하게 변환하여 데이터의 연속성을 보장합니다.
* **기본 텍스트 정규화 (Basic Normalization):** 단어 앞뒤의 불필요한 공백을 제거하고, 영문 대소문자를 하나로 통일하여 컴퓨터가 동일한 단어로 인식할 수 있도록 데이터의 모양을 다듬습니다.

#### 3-2. 형태소 분석 (Morphological Analysis): 문장의 속마음 읽기

한국어는 조사와 어미가 발달한 언어이므로, 단순한 단어 분리가 아닌 '의미의 최소 단위'인 형태소를 파악하는 것이 분석의 핵심입니다.

*
**Kiwi 형태소 분석 엔진 활용:** 최신 한국어 언어 모델을 도입하여, 띄어쓰기가 파괴되거나 은어가 섞인 복잡한 유튜브 댓글 문장에서도 단어의 경계를 정확하게 찾아냅니다.


*
**핵심 품사 필터링 (POS Filtering):** 전체 문장에서 여론 파악에 핵심이 되는 일반 명사(NNG)와 고유 명사(NNP)만을 정밀하게 추출합니다. 이를 통해 의미 없는 조사나 감탄사를 걸러내고 순도 높은 데이터만 남깁니다.


#### 3-3. 분석 품질 고도화 기법: 시각화를 통한 반복적 정제(Iterative Refinement) 🌟

이 시스템의 가장 큰 전문성은 분석을 한 번에 끝내지 않고, '1차 분석 ➔ 시각화 검증 ➔ 사전 업데이트 ➔ 재분석'으로 이어지는 정교한 피드백 루프를 갖추고 있다는 점입니다.

1. **시각화 기반 불순물 검증:** 1차 형태소 분석 결과를 바탕으로 워드클라우드나 단어 빈도표를 시각화하여, 기계가 놓친 불순물이나 잘못 쪼개진 단어를 사람의 눈으로 직접 교차 검증합니다.
2.
**동적 사전(Dynamic Dictionary) 연동:** 검증 과정에서 발견된 오류를 보정하기 위해 데이터베이스의 사전(Dictionary)을 지속적으로 업데이트합니다.


*
**사용자 정의 사전 (User Dictionary):** 신조어나 고유명사(예: '한동훈', '이재명')가 분절되는 것을 막기 위해 한 단어로 사전에 강제 등록합니다.


*
**표현 통일 사전 (Normalization Map):** 형태는 다르지만 같은 의미를 지닌 단어들(예: '국힘', '국힘당' ➔ '국민의힘')을 표준어로 매핑하여 여론 데이터가 분산되는 것을 방지합니다.


*
**불용어(Stopword) 처리:** 1차 시각화에서 지나치게 크게 나타나는 무의미한 단어(예: '영상', '유튜브', '생각' 등)를 블랙리스트에 추가하여 즉각적으로 걸러냅니다.


3.
**재분석(Re-run) 및 결과 확정:** 고도화된 사전을 바탕으로 데이터 베이스 내의 기존 실행 이력을 초기화하고 재분석을 수행하여, 완벽하게 정제된 최종 핵심어 토큰만을 도출해 냅니다.
