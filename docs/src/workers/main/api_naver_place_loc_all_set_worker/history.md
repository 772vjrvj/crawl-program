# 변경 이력

## 2026-07-18

### 수정

* 지역 선택 기준을 `시도`, `시군구`, `읍면동`으로 구분하여 수집할 수 있도록 수정
* 설정 화면에 지역 단위 선택 항목 추가
* 선택한 지역 단위에 맞게 전국 지역 목록을 가공하도록 수정
* 동일한 시도 또는 시군구가 반복되지 않도록 지역 목록 중복 제거
* 지역 단위와 키워드를 조합하여 네이버 플레이스 검색어를 생성하도록 수정

### 지역별 검색 기준

#### 시도

시도 단위로 검색한다.

```text
경기도 파크골프
서울특별시 파크골프
```

#### 시군구

시도와 시군구를 조합하여 검색한다.

```text
경기도 수원시 파크골프
서울특별시 강남구 파크골프
```

#### 읍면동

시도, 시군구, 읍면동을 조합하여 검색한다.

```text
경기도 수원시 영통구 망포동 파크골프
서울특별시 강남구 역삼동 파크골프
```

### 결정사항

* 지역 선택값은 `naver_loc` 설정으로 관리
* 기본 지역 단위는 `읍면동`으로 사용
* 선택한 지역 단위보다 하위 주소는 검색어에서 제외
* 지역과 키워드를 조합한 값을 전체 검색어로 저장
* 검색 결과의 중복 제거는 Place ID 기준으로 처리
* 동일한 Place ID가 여러 지역 또는 키워드에서 발견돼도 최초 한 건만 저장
* 상세 조회와 DB 저장이 성공한 Place ID만 중복 목록에 추가
* 상세 조회에 실패한 Place ID는 다른 검색어에서 다시 발견되면 재시도 가능

### 테스트

* 시도 선택 시 시도 단위 검색어 생성 확인
* 시군구 선택 시 시도 + 시군구 검색어 생성 확인
* 읍면동 선택 시 시도 + 시군구 + 읍면동 검색어 생성 확인
* 지역 목록 중복 제거 확인
* 지역과 키워드 조합 순회 확인
* 검색 지역 정보 DB 저장 확인

### 2026-04-11 — 느릿느릿크리스마스

| 구분 | 내용 |
|---|---|
| 유입 경로 | 크몽 |
| 크몽 ID | 느릿느릿크리스마스 |
| 가격 | 100,000원 |
| 작업 날짜 | 2026-04-11 |
| 작업 내용 | `파크골프` 키워드 전국 네이버 플레이스 수집 |
| 작업 범위 | 크롤링만 진행 |

#### 요청사항

`파크골프` 키워드로 전국 네이버 플레이스 정보를 수집한다.

요청 항목:

* 플레이스 ID
* 업종 분류 및 태그
* 대표 키워드
* 영업시간
* 휴무일
* 대표자 성함
* SNS 링크
* 리뷰 수
* 리뷰 평점
* 개업일 또는 첫 리뷰 등록일
* 시설 사진 URL

#### 수집 결과 및 제한사항

* 플레이스 ID 수집 가능
* 업종 분류 및 카테고리 수집 가능
* 대표 키워드 및 연관 키워드 수집 가능
* 영업시간 및 휴무 정보 수집 가능
* 홈페이지 및 SNS 링크 수집 가능
* 방문자 리뷰 수, 블로그 리뷰 수, 리뷰 평점 수집 가능
* 대표 이미지 및 시설 사진 URL 수집 가능
* 대표자 성함은 네이버 플레이스에서 확인할 수 없어 수집하지 못함
* 개업일은 플레이스 데이터에서 직접 제공되지 않는 경우 수집이 어려움
* 첫 리뷰 등록일은 별도 리뷰 전체 조회가 필요하므로 기본 수집 범위에서 제외

#### 결정사항

* 대표자 성함은 수집 불가 항목으로 안내
* 네이버 플레이스에서 직접 확인 가능한 정보만 수집
* 별도 시스템 개발 없이 크롤링 결과만 제공
* 전국 지역과 `파크골프` 키워드를 조합하여 수집
