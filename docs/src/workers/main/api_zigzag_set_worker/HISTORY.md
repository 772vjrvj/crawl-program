* 구글 스프레드시트 ID, 워크시트명, 인증 JSON 경로 설정 추가

* 수집 결과 컬럼 정의

    * 대표자

### 결정사항

* 상품 기준으로 상세정보를 조회하되 최종 저장은 판매자 기준으로 처리
* 판매자 중복키는 `업체명 + 대표자 + 전화번호` 조합 사용
* 동일 판매자가 여러 카테고리에서 발견돼도 최초 한 건만 유지
* URL과 상품ID는 내부 확인용으로 유지
* 스크롤 1회 분량을 1페이지로 처리하고 페이지당 약 50개 상품 조회
* 구글 시트 기존 데이터와 비교하여 신규 판매자만 추가하도록 구성
